🏢 Insurance Capability Map Vertical Flex Wrap (снизу вверх)

📐 Группы расположены в колонках с помощью flex-flow: column wrap и фиксированной высоты контейнера. При скрытии содержимого группы её высота уменьшается → соседние группы справа подтягиваются снизу вверх, занимая свободное место. Нажмите на любой заголовок, чтобы увидеть эффект.
⚡ Динамический расчёт высоты контейнера для оптимального заполнения. При сворачивании/разворачивании группы остальные перетекают вверх, заполняя пустоты.
